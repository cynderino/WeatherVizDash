Summary: Latitude vs. X

"What's the weather like as we approach the equator?" To answer that question, various weather factors including temperature, humidity, cloudiness, and wind speed for more than 500 cities was captured using the OpenWeatherMap API.

Python was utilized for analysis and Matplotlib for the visualizations.
